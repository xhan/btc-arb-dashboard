聚合报价看板 - 使用说明
欢迎使用聚合报价看板！这是一个轻量、高效的本地化工具，用于实时监控多链、多平台的加密货币报价，并提供自定义提醒功能。

一、 如何在本地运行项目
要成功在您的电脑上运行此项目，请遵循以下步骤。

前提条件:
您需要先在电脑上安装 Node.js。本项目依赖 Node.js 环境来运行后端服务。

下载地址: https://nodejs.org/ (推荐下载 LTS 版本)

安装 Node.js 时，请确保勾选了 npm package manager 选项。

首次安装与启动步骤:
1.解压项目文件

2.进入项目目录并打开终端:

Windows 快捷技巧(用这个就好): 打开解压后的项目文件夹，在顶部的地址栏（显示文件夹路径的地方）直接输入 cmd 然后按回车，即可在此处快速打开终端。

常规方法:

Windows: 搜索并打开 "命令提示符" 或 "PowerShell"。

macOS / Linux: 打开 "终端" 应用。

在终端里，使用 cd 命令进入您刚刚解压出来的文件夹。例如：
cd C:\Users\YourName\Desktop\quote-dashboard (请将路径替换为您的实际路径)

3.安装依赖 (仅首次需要):
在终端里输入以下命令并按回车。此命令会自动下载后端服务所需的所有依赖库。这个步骤只需要在第一次设置时执行一次。

npm install

等待命令执行完毕。

4.启动后端服务:
在终端里，输入以下命令并按回车：

npm start

提示: npm start 是推荐的启动方式。您也可以使用 node server.js 命令，效果是完全一样的。

如果看到 🚀 聚合报价后端服务正在 http://localhost:3000 上运行 的提示，说明后端已成功启动。
注意：使用时请不要关闭这个终端窗口，否则后端服务会停止。

5.打开前端页面:
回到您的项目文件夹，直接双击打开 index.html 文件，推荐谷歌浏览器。

后续日常使用:
当您第一次成功安装并运行过项目后，之后每天再想使用时，流程会非常简单：

进入项目文件夹，用快捷方式或常规方式打开终端。

直接运行 npm start 启动后端服务。

双击 index.html 文件。

无需再次执行 npm install 命令。

二、 如何跨设备同步配置
您在看板上添加的所有分区、报价监控和提醒规则都保存在 config.json 这个文件中。因此，要实现跨设备同步，您只需要同步这个文件即可。

同步步骤:
在源设备上:

找到项目文件夹。

复制其中的 config.json 文件。

在新设备上:

首先，按照 (一) 如何在本地运行项目 的所有步骤，在新设备上把整个项目环境搭建好。

在启动后端服务之前，将您从源设备复制过来的 config.json 文件，粘贴到新设备的项目文件夹中，覆盖掉原来的那个 config.json 文件。

现在，启动后端服务 (npm start)。

最后，用浏览器打开 index.html。

此时，您会发现新设备上的看板已经和您源设备上的配置完全一样了。

祝您使用愉快！
