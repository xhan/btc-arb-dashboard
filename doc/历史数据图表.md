# 交易对历史价格图表

现在项目中已经有历史数据快照.
需要个新功能, 可以知道 每个交易对的 价格走势(读取历史快照).

帮我做个新的页面,可以查看历史曲线. 路径叫 /charts. 
同时, 看板主页导航上也加上链接,可以跳转.

大致功能:

可以根据这个格式
(ETH) GHO -> USDC  

输入 链, token1 , token 2 .

为了方便输入,最好有自动完成功能,方便快速选择.

图表用tradingview 的简单图表.

时间只需要最近两小时的数据.

可以添加多张图表,也就是说,可以添加多个交易对.

还需要的功能:

比如,在 套利路径 UI 面板上, 点某个机会 的图表链接(可以是个小箭头表示),
可以打开 "交易对历史价格图表",
并加载 机会 中的几个路径 的历史图表.


# 注意事项

一定不要因为引入新功能,让项目代码越来越臃肿.
分模块化,对应的功能单独放.
你要以顶级程序员的思维和视角来添加功能.
